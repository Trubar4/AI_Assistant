Abweichung des Vertical Line Finder auf Null setzen

Sicherstellen, dass folgende Voraussetzungen erfüllt sind:

❑

Maschine hat keine Seitenneigung (Weitere Informationen siehe: „Neigungsanzeige“).

❑

Last ist gehoben.

❑

Last ist freihängend.

❑

Last pendelt nicht.

❑

Gewicht der Last ist minimum 2 t (4,409 lb) x Anzahl der Einscherungen.

Die Abweichung des Vertical Line Finder muss bei jeder Änderung der Einscherung oder der Auslegerkonfiguration auf Null gesetzt werden.

Taste Bildschirmseite Einstellungen am Bildschirm drücken.

Bildschirmseite Einstellungen wird am Bildschirm eingeblendet.

Taste Abweichung des Vertical Line Finder auf Null setzen am Bildschirm länger als 1 Sekunde drücken.

Abweichung des Vertical Line Finder wird auf Null gesetzt.
