Reuse in another GitHub project

Three options:

  1. Submodule: git submodule add <design-system-repo> design-system
  2. npm package: publish design-system/ as @your-org/lds and import the CSS
  3. CDN: host lds.css on any static host and link via <link rel="stylesheet">
